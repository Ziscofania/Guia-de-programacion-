# Guía de Programación

La **Guía de Programación** es una aplicación web diseñada para ofrecer un espacio de aprendizaje y consulta para programadores. Su objetivo es facilitar el acceso a información práctica, ejemplos, conceptos y recursos útiles para estudiantes y desarrolladores interesados en mejorar sus habilidades técnicas.  

---

## Características principales

- Interfaz desarrollada con **HTML, CSS y JavaScript puro**.  
- Estructura modular y fácil de mantener.  
- Navegación intuitiva con menús organizados por temas.  
- Compatibilidad con navegadores modernos.  
- Diseño adaptable a diferentes resoluciones de pantalla.  

---

## Objetivo del proyecto

El propósito de este proyecto es brindar una plataforma educativa clara, funcional y visualmente atractiva que permita a los usuarios acceder a guías y recursos sobre programación de forma rápida y estructurada.  

---

## Tecnologías utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Gestión del proyecto:** Git y GitHub
- **Pruebas:** Jest (configurado con entorno `jsdom`)
- **Docker:** Contenedor para ejecutar la aplicación de forma aislada y reproducible en cualquier entorno


## Instalación y ejecución

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/Ziscofania/Guia-de-programacion-.git
   ```
   ```bash
   cd Guia-de-programacion
    ```
2. **Instalar dependencias en las carpetas Backend y Frontend**

    ```bash
    npm install
    ```
