Introducción al Desarrollo Web

El desarrollo web frontend se compone de tres tecnologías fundamentales que trabajan juntas:

HTML

Lenguaje de marcado para estructurar el contenido de las páginas web.

Versión actual: HTML5

CSS

Lenguaje de estilos para definir la presentación visual del contenido HTML.

Versión actual: CSS3

JavaScript

Lenguaje de programación para añadir interactividad y comportamiento dinámico.

Versión actual: ECMAScript 2022

¿Cómo trabajan juntas?

HTML es como la estructura de una casa (paredes, techos), CSS es la decoración y pintura, y JavaScript es la electricidad y sistemas que hacen la casa funcional.

Evolución de las tecnologías web

1991
Tim Berners-Lee crea HTML
1996
Aparece CSS
1997
JavaScript se estandariza como ECMAScript
2014
HTML5 y CSS3 se completan
2015
ES6 (gran actualización de JavaScript)

HTML Básico

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el significado del contenido.

Estructura básica de un documento HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página</title>
  </head>
  <body>
    <h1>Mi Primer Título</h1>
    <p>Mi primer párrafo.</p>
  </body>
</html>

Elementos HTML esenciales

Ejemplo HTML
<h1>Título principal</h1>
<h2>Subtítulo</h2>
<p>Párrafo de texto.</p>
<a href="https://ejemplo.com">Enlace</a>
<img src="imagen.jpg" alt="Descripción">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
Resultado en navegador

Resultado:

Título principal

Subtítulo

Párrafo de texto.

Enlace

[Imagen]
  • Item 1
  • Item 2

Importante: Todos los elementos HTML deben tener etiquetas de apertura y cierre (ej: <p></p>), excepto los elementos vacíos como <img> que se cierran con /> en XHTML.

HTML5 Semántico

HTML5 introdujo elementos semánticos que describen mejor el significado del contenido, ayudando a los motores de búsqueda y tecnologías de accesibilidad.

Estructura semántica típica

<header>
  <!-- Logo, navegación principal -->
</header>
<nav>
  <!-- Menús de navegación -->
</nav>
<main>
  <article>
    <!-- Contenido independiente -->
  </article>
  <aside>
    <!-- Contenido relacionado -->
  </aside>
</main>
<footer>
  <!-- Información de pie -->
</footer>

Elementos semánticos clave

  • <header>: Cabecera de la página o sección
  • <nav>: Bloque de enlaces de navegación
  • <main>: Contenido principal único en la página
  • <article>: Contenido independiente (post, noticia)
  • <section>: Sección temática de contenido
  • <aside>: Contenido relacionado indirectamente
  • <footer>: Pie de página o sección
  • <figure> y <figcaption>: Para imágenes con descripción

SEO Tip: Usar elementos semánticos correctamente puede mejorar el posicionamiento en buscadores, ya que ayudan a los robots a entender la estructura de tu contenido.

Formularios HTML

Los formularios permiten a los usuarios interactuar con la página web, enviando datos al servidor.

Estructura básica de un formulario

Código HTML
<form action="/procesar" method="POST">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje" name="mensaje"></textarea>

  <button type="submit">Enviar</button>
</form>
Resultado

Formulario de contacto:

Tipos de inputs comunes

  • text: Campo de texto básico
  • password: Para contraseñas (oculta texto)
  • email: Validación básica de email
  • number: Solo números
  • date: Selector de fecha
  • checkbox: Casillas de verificación
  • radio: Botones de opción única
  • file: Subida de archivos
  • submit: Botón de envío

Seguridad: Nunca confíes solo en la validación del lado del cliente. Siempre valida los datos en el servidor también.

CSS Básico

CSS (Cascading Style Sheets) es el lenguaje que controla el aspecto visual de los documentos HTML.

Sintaxis básica de CSS

selector {
  propiedad: valor;
  otra-propiedad: valor;
}

Formas de aplicar CSS

Estilos en línea
<p style="color: red;">
  Texto rojo
</p>
Estilos internos
<style>
p {
  color: red;
}
</style>
Hoja de estilos externa
<link rel="stylesheet" href="estilos.css">

Contenido de estilos.css:

p {
  color: red;
}

Mejor práctica: Usa siempre hojas de estilo externas para separar el contenido de la presentación y facilitar el mantenimiento.

Selectores básicos

/* Selector de elemento */
p {
  color: blue;
}

/* Selector de clase */
.destacado {
  font-weight: bold;
}

/* Selector de ID */
#header {
  background: #333;
}

/* Selector de atributo */
input[type="text"] {
  border: 1px solid #ccc;
}

Diseño con CSS

CSS ofrece varios métodos para controlar el diseño y posicionamiento de elementos en la página.

Modelo de caja (Box Model)

.caja {
  width: 300px;
  height: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 30px;
  box-sizing: border-box; /* Incluye padding y border en el width */
}

Sistemas de layout

Flexbox

Diseño unidimensional (filas o columnas) ideal para componentes y diseños pequeños.

.contenedor {
  display: flex;
  justify-content: center;
  align-items: center;
}

CSS Grid

Sistema bidimensional (filas y columnas) para diseños complejos de página completa.

.contenedor {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 20px;
}

Position

Posicionamiento exacto de elementos (relative, absolute, fixed, sticky).

.elemento {
  position: absolute;
  top: 20px;
  left: 50%;
}

Consejo: Usa Flexbox para componentes y diseños en una dimensión, y CSS Grid para layouts de página completa en dos dimensiones.

CSS3 Avanzado

CSS3 introdujo numerosas características avanzadas para crear diseños más sofisticados.

Animaciones y transiciones

Transición CSS
.boton {
  background: blue;
  transition: background 0.3s ease;
}

.boton:hover {
  background: darkblue;
}
Animación CSS
@keyframes desvanecer {
  from { opacity: 0; }
  to { opacity: 1; }
}

.elemento {
  animation: desvanecer 2s infinite;
}

Transformaciones

.elemento {
  transform: rotate(15deg) scale(1.1) translateX(20px);
  transform-origin: center center;
}

Variables CSS (Custom Properties)

:root {
  --color-primario: #3b82f6;
  --espaciado: 20px;
}

.elemento {
  color: var(--color-primario);
  padding: var(--espaciado);
}

Diseño Responsive

El diseño responsive permite que tus páginas se adapten a diferentes tamaños de pantalla.

Viewport meta tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

/* Estilos base */
.contenedor {
  width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
  .contenedor {
    width: 750px;
  }
}

/* Escritorio */
@media (min-width: 992px) {
  .contenedor {
    width: 970px;
  }
}

Enfoques responsive

  • Mobile First: Diseña primero para móviles y luego añade estilos para pantallas más grandes
  • Unidades relativas: Usa %, vw, vh, rem, em en lugar de px fijos
  • Imágenes flexibles: max-width: 100%; height: auto;
  • Grid y Flexbox: Son intrínsecamente responsive

Puntos de ruptura comunes: 576px (móviles), 768px (tablets), 992px (pequeños escritorios), 1200px (escritorios grandes).

JavaScript Básico

JavaScript es el lenguaje de programación que permite añadir interactividad a las páginas web.

Sintaxis básica

// Variables
let mensaje = "Hola Mundo";
const PI = 3.1416;
var edad = 25; // Menos usado en ES6+

// Funciones
function saludar(nombre) {
  return `Hola ${nombre}`;
}

// Arrow function (ES6+)
const sumar = (a, b) => a + b;

// Condicionales
if (edad >= 18) {
  console.log("Mayor de edad");
} else {
  console.log("Menor de edad");
}

// Bucles
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// Arrays
const frutas = ['manzana', 'banana'];
frutas.push('naranja');

Tipos de datos

  • Primitivos: String, Number, Boolean, Null, Undefined, Symbol, BigInt
  • Objetos: Object, Array, Function, Date, etc.

Cuidado: JavaScript tiene coerción de tipos que puede llevar a resultados inesperados (ej: "5" + 3 = "53" pero "5" - 3 = 2).

Manipulación del DOM

El DOM (Document Object Model) es la representación en objetos del documento HTML que JavaScript puede manipular.

Seleccionar elementos

// Selección tradicional
const elemento = document.getElementById('miId');
const elementos = document.getElementsByClassName('miClase');
const tags = document.getElementsByTagName('p');

// Métodos modernos (querySelector)
const el = document.querySelector('#miId'); // ID
const els = document.querySelectorAll('.miClase'); // Clase

Manipular elementos

// Cambiar contenido
elemento.textContent = 'Nuevo texto';
elemento.innerHTML = '<strong>Texto</strong> con HTML';

// Cambiar estilos
elemento.style.color = 'red';
elemento.style.backgroundColor = '#eee';

// Añadir/remover clases
elemento.classList.add('nueva-clase');
elemento.classList.remove('vieja-clase');
elemento.classList.toggle('clase-activa');

// Crear nuevo elemento
const nuevoDiv = document.createElement('div');
document.body.appendChild(nuevoDiv);

Eventos

// Método tradicional
elemento.addEventListener('click', function(evento) {
  console.log('Clic en:', evento.target);
});

// Eventos comunes
// click, dblclick, mouseenter, mouseleave
// keydown, keyup, keypress
// submit, change, input, focus, blur
// load, resize, scroll

ES6+ Moderno

ECMAScript 2015 (ES6) introdujo importantes mejoras a JavaScript. Estas son algunas características clave:

Variables let/const

// let permite reasignación
let nombre = 'Juan';
nombre = 'Carlos'; // OK

// const es para constantes
const PI = 3.1416;
PI = 3; // Error!

Arrow functions

// Función tradicional
function sumar(a, b) {
  return a + b;
}

// Arrow function equivalente
const sumar = (a, b) => {
  return a + b;
};

// Forma más corta (return implícito)
const sumar = (a, b) => a + b;

Template literals

const nombre = 'Juan';
const edad = 30;

// Antes
const mensaje = 'Hola ' + nombre + ', tienes ' + edad + ' años.';

// Con template literals
const mensaje = `Hola ${nombre}, tienes ${edad} años.`;

Destructuring

// Arrays
const numeros = [1, 2, 3];
const [primero, segundo] = numeros;

// Objetos
const persona = { nombre: 'Juan', edad: 30 };
const { nombre, edad } = persona;

Módulos (import/export)

// Exportar en modulo.js
export const PI = 3.1416;
export function sumar(a, b) { return a + b; }

// Importar en otro archivo
import { PI, sumar } from './modulo.js';
