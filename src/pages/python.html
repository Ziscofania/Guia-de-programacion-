Introducción a Python

Python es un lenguaje de programación interpretado, de alto nivel y multipropósito, creado por Guido van Rossum en 1991. Es conocido por su sintaxis clara y legible que favorece la productividad del desarrollador.

Características principales

  • Multiplataforma: Funciona en Windows, macOS y Linux
  • Multiparadigma: Soporta programación orientada a objetos, funcional e imperativa
  • Tipado dinámico: No es necesario declarar el tipo de variable
  • Gran ecosistema: Miles de librerías para casi cualquier necesidad
  • Comunidad activa: Una de las comunidades más grandes y activas

Sabías que: Python es el lenguaje más popular según el Índice TIOBE y es ampliamente usado en campos como inteligencia artificial, ciencia de datos, desarrollo web y automatización.

Instalación de Python

Python viene preinstalado en la mayoría de sistemas Linux y macOS, pero para Windows o para obtener la última versión necesitarás instalarlo manualmente.

Windows

  1. Descarga el instalador desde python.org
  2. Ejecuta el instalador
  3. Marca la opción "Add Python to PATH" (IMPORTANTE)
  4. Selecciona "Install Now"

macOS

Opción 1: Usa Homebrew (recomendado):

brew install python

Opción 2: Descarga el instalador desde python.org

Linux (Debian/Ubuntu)

Python 3 viene preinstalado en la mayoría de distribuciones. Para asegurarte:

sudo apt update sudo apt install python3 python3-pip python3-venv

Verificar instalación

Abre una terminal y ejecuta:

python --version # o python3 --version

Deberías ver algo como: Python 3.9.7

Configuración inicial

Variables de entorno PATH

Asegúrate que Python y pip están en tu PATH para poder usarlos desde cualquier directorio.

Windows

Si marcaste "Add Python to PATH" durante la instalación, ya debería estar configurado. Si no:

  1. Busca "Variables de entorno" en el menú de inicio
  2. Selecciona "Editar las variables de entorno del sistema"
  3. Haz clic en "Variables de entorno..."
  4. En "Variables del sistema", busca "Path" y haz clic en "Editar"
  5. Agrega las rutas a Python y Scripts (normalmente C:\Python39 y C:\Python39\Scripts)

macOS/Linux

Normalmente se agrega automáticamente. Verifica con:

which python3 which pip3

Configurar pip

Pip es el gestor de paquetes de Python. Actualízalo y configura un espejo más rápido si es necesario:

python -m pip install --upgrade pip # Opcional: Configurar mirror más rápido (ejemplo para China) pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

Entornos virtuales

Los entornos virtuales permiten aislar las dependencias de cada proyecto. Es una buena práctica crear uno para cada proyecto.

Crear un entorno virtual

# Crear entorno python -m venv mi_entorno # Activar (Windows) mi_entorno\Scripts\activate # Activar (macOS/Linux) source mi_entorno/bin/activate

Comandos útiles

# Desactivar entorno deactivate # Listar paquetes instalados pip list # Guardar dependencias a archivo pip freeze > requirements.txt # Instalar desde requirements.txt pip install -r requirements.txt

Tip: Usa python -m venv en lugar de virtualenv ya que viene incluido en Python 3.3+.

Estructuras de datos

Listas

# Creación frutas = ['manzana', 'banana', 'naranja'] # Acceso primera_fruta = frutas[0] # 'manzana' # Métodos comunes frutas.append('uva') # Añade al final frutas.insert(1, 'mango') # Inserta en posición frutas.remove('banana') # Elimina elemento

Tuplas

# Inmutables coordenadas = (10, 20) x, y = coordenadas # Desempaquetado

Diccionarios

persona = { 'nombre': 'Ana', 'edad': 30, 'ciudad': 'Madrid' } # Acceso nombre = persona['nombre'] # Métodos persona['trabajo'] = 'Ingeniera' # Añadir/actualizar del persona['edad'] # Eliminar

Sets

# Colección no ordenada y sin duplicados numeros = {1, 2, 3, 3, 4} # {1, 2, 3, 4} # Operaciones de conjuntos a = {1, 2, 3} b = {3, 4, 5} union = a | b # {1, 2, 3, 4, 5} interseccion = a & b # {3}

Python tiene un rico ecosistema de librerías. Estas son algunas de las más importantes:

NumPy

pip install numpy

Pandas

Manipulación y análisis de datos estructurados.

pip install pandas

Matplotlib

Peticiones HTTP sencillas y elegantes.

pip install requests

BeautifulSoup

Web scraping y parsing de HTML/XML.

Flask/Django

Frameworks para desarrollo web.

pip install flask django

Gestión de paquetes con pip

Pip es el sistema de gestión de paquetes estándar de Python. Permite instalar y administrar librerías adicionales.

Comandos esenciales

# Instalar un paquete pip install nombre_paquete # Instalar versión específica pip install nombre_paquete==1.2.3 # Actualizar un paquete pip install --upgrade nombre_paquete # Desinstalar pip uninstall nombre_paquete # Listar paquetes instalados pip list # Mostrar información de un paquete pip show nombre_paquete # Guardar dependencias pip freeze > requirements.txt # Instalar desde requirements.txt pip install -r requirements.txt

Precaución: Evita instalar paquetes globalmente (sin entorno virtual) para prevenir conflictos entre proyectos.

Entornos de desarrollo

Editores y IDEs recomendados

  • VS Code: Ligero y altamente configurable con extensión Python
  • PyCharm: IDE completo especializado para Python (versión Community gratuita)
  • Jupyter Notebook: Ideal para ciencia de datos y análisis exploratorio
  • Sublime Text: Editor rápido con buen soporte para Python
  • Vim/Emacs: Para usuarios avanzados que prefieren terminal

Configuración recomendada para VS Code

  1. Instala VS Code desde code.visualstudio.com
  2. Abre la paleta de comandos (Ctrl+Shift+P) y busca "Extensions: Install Extensions"
  3. Instala la extensión "Python" de Microsoft
  4. Instala "Pylance" para mejor autocompletado
  5. Configura el linter (pylint o flake8) y formateador (autopep8 o black)

Productividad: Aprende los atajos de teclado de tu editor favorito para aumentar significativamente tu velocidad de desarrollo.

Buenas prácticas en Python

Estilo de código (PEP 8)

  • Usa 4 espacios por nivel de indentación
  • Líneas de máximo 79 caracteres
  • Nombres de variables en snake_case: mi_variable
  • Nombres de clases en CamelCase: MiClase
  • Constantes en MAYÚSCULAS: MI_CONSTANTE
  • Espacios alrededor de operadores y después de comas

Consejos generales

  • Usa docstrings para documentar funciones y clases
  • Divide tu código en módulos lógicos
  • Escribe pruebas unitarias (unittest, pytest)
  • Maneja excepciones adecuadamente
  • Evita el código repetido (DRY - Don't Repeat Yourself)
  • Usa type hints para mejorar la legibilidad (Python 3.5+)

Ejemplo de función bien escrita

def calcular_promedio(numeros: list[float]) -> float: """Calcula el promedio de una lista de números. Args: numeros: Lista de números para calcular el promedio. Returns: El promedio de los números. Raises: ValueError: Si la lista está vacía. """ if not numeros: raise ValueError("La lista no puede estar vacía") return sum(numeros) / len(numeros)
